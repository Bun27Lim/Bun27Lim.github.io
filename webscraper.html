Yahoo Finance
Web Scraper

Purpose

The purpose of this webscraper is to automate the extraction of data from Yahoo Finance to gather information of the top 25 stocks. I built this project to practice list and dictionary comprehension to clean and make data more usable. Data can then be exported to a data visualization software such as Power BI or Tableau.

Objective

  • Data Collection
  • Data Manipulation
  • Webscraping Compliance and Ethics
  • Time Efficient Automation

Insights

  • List and Dictionary manipulation in python for data comprehension
  • Learning about webscraping compliance and ethics with robots.txt webpage that permits which webpage is allowing webscraping
  • Another way to gather data for analytical purposes
  • An introduction to python as a programming language
  • An introduction to big python libraries such as beautifulsoup4 and pandas
  • Parsing of the HTML structure
