About

The game is called PolyPals. This is a game developed in C/C++ using the SDL library to provide a hardware abstraction layer for computer multimedia hardware components. A game loop is implemented to loop each frame of the game. This is a class team project of consisting for 3 members.

My Role

One of my many contribution to this project was the implementation of the game loop, which is necessary for the game to function. For most games a loop is used for rendering, event triggers, event listening, etc to constantly run the game. Another one of my major contributions was the character movement. By implementing a system of event listening to input, I can use WASD keys for translational movement and QE keys for rotational movement.
The project took about one month to complete.

Game objective

Inspired by the game show "Hole in the Wall." The objective of the game is to fit the shape in the hole as fast as possible by moving and rotating. The more faster and accurate you are the higher the score. At the beginning of the round you start with 1000 points and as the game loops the score goes down until the shape is placed in the hole.
